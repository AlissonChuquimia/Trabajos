# 🌡️ Predicción de Temperatura en Bolivia (2026–2030)

Pronóstico de la **temperatura media** por ciudades de Bolivia con series temporales, a partir de
datos históricos **1990–2026** (fuente: INE / SENAMHI).

## 🧪 Contenido
- Modelo **SARIMA** para el pronóstico mensual 2026–2030 por ciudad.
- Predicción **diaria** para El Alto (junio 2026).
- Exportación de resultados y guía de **dashboard en Tableau**.

## 📁 Archivos
- `Bolivia_Temperatura_Prediccion_2026_2030.ipynb` — notebook principal.
- `*.xlsx` / `*.csv` — datos históricos y predicciones generadas.

## 🛠️ Stack
Python · Pandas · statsmodels (SARIMA) · Matplotlib · Tableau
