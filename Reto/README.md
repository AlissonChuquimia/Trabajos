# 🛒 Análisis de Datos de E-commerce (Olist)

Análisis integral del dataset público de e-commerce **Olist** (Brasil): manipulación de datos,
estadística, visualización estática e interactiva e interpretación. Desarrollado en el **Bootcamp
de Ciencia de Datos (NAO Digital)**.

## 🧪 Contenido
- `RETO_1_pandas.ipynb` — integración de tablas con Pandas.
- `RETO_2_*` — estadística, probabilidad y visualizaciones.
- `RETO_3_*` — visualizaciones estáticas (Matplotlib / Seaborn) y entregables.
- `RETO_4_visualizaciones_interactivas.ipynb` — Plotly, incluyendo mapas geográficos.

## 📦 Datos
Dataset **Brazilian E-Commerce (Olist)** de Kaggle. Los archivos grandes **no se incluyen** (superan
el tamaño recomendado). Descárgalos de
<https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce>.

## 🛠️ Stack
Python · Pandas · Matplotlib · Seaborn · Plotly
