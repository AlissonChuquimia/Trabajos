# 👥 Predicción de Rotación de Personal (RR.HH.)

Modelos de clasificación para predecir qué empleados tienen mayor probabilidad de **renunciar**,
como apoyo a la retención de talento.

## 🧪 Enfoque
- EDA y limpieza del dataset de RR.HH.
- Comparación de **Regresión Logística**, **Random Forest** y **Deep Learning (red neuronal)**.
- Evaluación con accuracy, matriz de confusión y reporte de clasificación.

## 📊 Resultados
- Mejor desempeño ~**83 % de accuracy**.

## 🛠️ Stack
Python · scikit-learn · TensorFlow / Keras · Pandas
